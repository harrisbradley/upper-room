# MVP Product Requirements (v1)

## Goal

Enable a new leader to:

1. Create a Bible study  
2. Share a join link  
3. Plan sessions  
4. Post recaps  

All within a **simple, peaceful experience**.

## North Star Metric

**% of visitors who create a study and generate a share link in under 2 minutes.**

---

## Core User Roles

### Leader
- Creates study
- Plans sessions
- Leads discussion
- Posts recap

### Participant
- Joins study
- Prepares with reading/questions
- Reviews recap

---

## Core User Flows

### 1. Create
Landing → Quick Create → Study created → Share link shown

### 2. Join
Join link → View study → Join → Become member

### 3. Plan
Leader dashboard → Edit session passage + questions

### 4. Recap
Leader posts summary → Members review later

---

## Features IN (v1)

- Anonymous create flow  
- Study creation (title, schedule, meeting mode, start date)  
- Shareable join link  
- Join page + membership  
- Sessions list  
- Edit session:
  - Scripture reference  
  - Discussion questions  
  - Leader notes  
- Post-session recap:
  - Summary  
  - Key takeaways  
  - Prayer intentions  

---

## Features OUT (later)

- Attendance tracking  
- Chat/messaging  
- Curriculum marketplace  
- AI-generated questions  
- Parish dashboards  
- Payments/donations  
- Deep analytics  

---

## Non-Goals

- Replace church websites  
- Provide theological authority  
- Become a social network  

This is a **ministry tool**, not a platform for attention.
